学了前端还是什么都不会的菜鸡

id选择器

ID 属性不要以数字开头，在某些浏览器中会失效。

1
2
3
4
5
6
7
8

#para1
{
	text-align:center;
	color:red;
} 

<p id="para1">Hello World!</p>
<p>这个段落不受该样式的影响。</p>

class选择器

描述某一个类的样式，在多个元素中生效

1
2
3
4
5
6
7

.center
{
	text-align:center;
}

<h1 class="center">标题居中</h1>
<p class="center">段落居中。</p>

也可以指定作用在某些特定元素上

1
2
3
4
5

只在段落上生效
p.center
{
	text-align:center;
}

注意类名的第一个字符不能使用数字。

创建

  • 外部样式表 拓展名.css 例子：

    1
2
3

    hr {color:sienna;}
p {margin-left:20px;}
body {background-image:url("/images/back40.gif");}

    引用：

    1
2
3

    <head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

  • 内部样式表 在头部用<style></style>定义

  • 内联样式 只应用在某一元素上，用style属性

    1

    <p style="color:sienna;margin-left:20px">这是一个段落。</p>

    存在多个样式文件时，优先级： 内联样式 > 内部样式 > 外部样式 > 浏览器默认样式

背景

  • background-color

    1
2
3

    h1 {background-color:#6495ed;}
p {background-color:#e0ffff;}
div {background-color:#b0c4de;}

  • background-image

    1

    background-image:url('paper.gif');

  • background-repeat 默认图像在垂直和水平方向密铺，可以更改密铺的方向：

    1
2

    background-image:url('gradient2.png');
background-repeat:repeat-x;

    又或者禁止密铺：

    1
2

    background-image:url('img_tree.png');
background-repeat:no-repeat;

  • background-attachment 固定背景图像的位置

    1
2
3

    background-image:url('smiley.gif');
background-repeat:no-repeat;
background-attachment:fixed;

  • background-position 改变图像位置

    1
2
3

    background-image:url('img_tree.png');
background-repeat:no-repeat;
background-position:right top;

    可选的位置包括[left,right,center]x[top,center,buttom]，也可以用x% y%表示

以上属性可以统一写在background属性中。用空格分隔，不要求顺序。

文本

  • 颜色

    1
2
3

    body {color:red;}
h1 {color:#00ff00;}
p.ex {color:rgb(0,0,255);}

  • 对齐

    1
2
3

    h1 {text-align:center;}
p.date {text-align:right;}
p.main {text-align:justify;}

    当text-align设置为”justify”，每一行被展开至宽度相等，左，右外边距对齐。

  • 修饰 text-decoration包括text-decoration-line, text-decoration-color和 text-decoration-style三个属性。可以定义文本上、中、下的划线。 比如：

    1

    text-decoration: underline wavy red;

  • 转换

    1
2
3

    p.uppercase {text-transform:uppercase;}
p.lowercase {text-transform:lowercase;}
p.capitalize {text-transform:capitalize;}

  • 缩进

    1

    p {text-indent:50px;}

  • 阴影

    1

    h1 {text-shadow:2px 2px #FF0000;}

  • 文本间距 对于英语，调整同一个单词的字母间的距离；对于中文，即文字间距。

    1
2

    h1 {letter-spacing:2px;}
h2 {letter-spacing:-3px;}

  • 单词间距

  • 对于中文无效

    1

    p{ word-spacing:30px; }
