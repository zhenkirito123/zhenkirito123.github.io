脚本必须位于 <script> 与 </script> 标签之间,可被放置在 HTML 页面的 <body> 和 <head> 部分中。

若使用外部 .js 文件，注意文件中不出现 <script> 标签，使用 <script src="myScript.js"></script> 引用。

输出

  • 使用 window.alert() 弹出警告框。

  • 使用 document.write() 方法将内容写到 HTML 文档中。 如果在文档已完成加载后执行 document.write，整个 HTML 页面将被覆盖。

document.getElementById("demo") 是使用 id 属性来查找 HTML 元素的 JavaScript 代码 。 - 使用 console.log() 写入到浏览器的控制台。

字面量

常量 - 数字 - 字符串 用单引号和双引号皆可 - 数组 如 [40,100,20] - 对象 如 {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"} - 函数 如 function myFunction(a, b) { return a * b;}

变量

用 var 定义。注意 Js 使用动态类型。 重新声明 JavaScript 变量，该变量的值不会丢失。 - 数字 - 字符串 - 可以使用索引位置来访问字符串中的每个字符，如 var character = carname[7]; - 可以在字符串中添加转义字符 \ 来使用引号等特殊字符。 - 使用 .length 获取长度。 - 布尔 - 数组

比较

=== / !== 为绝对相等，即数据类型与值都必须相等。 == / != 只比较转化成同一类型后的值。

使用标签配合 continue , break 使用，可以跳出多重循环。 注意 break 可用于跳出任何 JavaScript 代码块，而 continue 只能应用在循环语句中。
