颜色

由一个十六进制符号来定义，这个符号由红色、绿色和蓝色的值组成（RGB）。每种颜色的最小值是0（十六进制：#00）。最大值是255（十六进制：#FF）。

有141个标准颜色名，可以参考此处。

写法：#0f0f0f 或者 rgb(0,255,0) 。

拓展：增加 alpha 透明度，0表示全透明。 写法：rgba(255,255,0,0.25)
